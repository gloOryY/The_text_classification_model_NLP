# Text Sentiment Classification (Movie Reviews)
Проект по бинарной классификации текстов: модель определяет, является ли отзыв на фильм **позитивным** или **негативным** по содержимому текста.

## Описание проекта
- Входные данные: файл `moviereviews.csv` с колонками:
  - `review` – текст отзыва
  - `label` – метка класса (`pos` или `neg`)
- Цель: обучить модель, которая по тексту отзыва предсказывает его тональность.
- Подход:
  - Предобработка текстов
  - Векторизация с помощью **TF‑IDF**
  - Обучение линейного SVM (**LinearSVC**) для классификации
  - Объединение шагов в единый **Pipeline** для удобства и предотвращения утечки данных.

## Используемый стек
- Библиотеки:
  - `pandas` — загрузка и обработка данных
  - `scikit-learn`:
    - `train_test_split` — разбиение на train/test
    - `TfidfVectorizer` — преобразование текста в числовые признаки
    - `LinearSVC` — модель классификации
    - `Pipeline` — объединение шагов обработки и модели
    - `classification_report`, `ConfusionMatrixDisplay` — оценка качества модели
  - `matplotlib`, `seaborn` — визуализация

## Структура проекта
- `the_text_classification_model.ipynb` — основной ноутбук с кодом:
  - загрузка датасета
  - очистка текстов (удаление строк с пустыми/пробельными отзывами)
  - исследовательский анализ (распределение классов, длина отзывов и т.п.)
  - построение признаков с помощью TF‑IDF
  - обучение модели с помощью Pipeline
  - оценка качества модели (accuracy, precision, recall, f1, confusion matrix)
- `moviereviews.csv` — датасет с отзывами.

## Основные шаги в ноутбуке
1. **Загрузка данных**
   - Чтение `moviereviews.csv` в `DataFrame`.
   - Просмотр структуры данных и примеров отзывов.

2. **Очистка данных**
   - Удаление строк, где `review` состоит только из пробелов/пустых символов.
   - Проверка пропусков и корректности меток.

3. **Разделение на обучающую и тестовую выборки**
   - `X = df['review']`, `y = df['label']`
   - Разделение на `X_train`, `X_test`, `y_train`, `y_test`.

4. **Построение Pipeline**
   - Создание конвейера:
     - Шаг `tfidf`: `TfidfVectorizer()` — превращает тексты в TF‑IDF векторы.
     - Шаг `svc`: `LinearSVC()` — обучает линейный классификатор на этих векторах.
   - Обучение: `pipe.fit(X_train, y_train)`
   - Предсказание: `pipe.predict(X_test)`

5. **Оценка модели**
   - Получение предсказаний на тесте.
   - Вывод `classification_report` (precision, recall, f1-score).
   - Построение матрицы ошибок через `ConfusionMatrixDisplay`.
